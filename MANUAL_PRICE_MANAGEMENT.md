# Функционал ручного управления ценами

## Обзор изменений

Добавлена возможность **ручного управления ценами** как через веб-интерфейс, так и через Telegram-бота. Это позволяет устанавливать актуальные цены, когда интеграция с Avito ещё не настроена или требуется быстрая коррекция.

---

## 🌐 Веб-интерфейс

### Новая страница: `/manage-prices`

**URL:** `http://localhost:8000/manage-prices`  
**Доступ:** Требуется авторизация (логин/пароль)

### Возможности:

1. **Просмотр всего прайс-листа**
   - Таблица со всеми товарами (ID, название, объем, цена)

2. **Редактирование цены**
   - Поле ввода рядом с каждой позицией
   - Кнопка 💾 для быстрого сохранения

3. **Добавление нового товара**
   - Форма вверху страницы
   - Поля: Название масла, Объем (мл), Цена (руб)

4. **Удаление позиции**
   - Кнопка 🗑️ для каждой позиции
   - Подтверждение перед удалением

## 🤖 Telegram-бот

### Новая команда администратора: `✏️ Управление ценами`

**Доступ:** Через админ-панель (`/admin`)

### Возможности:

1. **Просмотр списка товаров**
   - Отображаются первые 20 позиций
   - Формат: `Название (Объем) - Цена₽`

2. **Редактирование цены**
   - Нажмите на товар в списке
   - Бот попросит ввести новую цену
   - Пример: `4500` или `4500.50`

3. **Добавление нового товара**
   - Кнопка `➕ Добавить новый товар`
   - Введите в формате: `Название, Объем, Цена`
   - Пример: `Motul 8100 X-cess, 5000, 4500`

4. **Навигация**
   - Кнопка `🔙 В меню админа` для возврата

### Пример диалога:

```
Админ: ✏️ Управление ценами

[Список товаров с кнопками]
• Motul 8100 X-cess (5000мл) - 4500₽
• Castrol Edge (4000мл) - 3800₽
[➕ Добавить новый товар]
[🔙 В меню админа]

(Пользователь нажимает на товар)

Бот: ✏️ Редактирование цены

Товар: Motul 8100 X-cess
Объем: 5000мл
Текущая цена: 4500₽

Введите новую цену:

(Пользователь вводит: 4600)

Бот: ✅ Цена обновлена на 4600₽
```

## 🚀 Как использовать

### Через веб-интерфейс:

1. Запустите приложение: `python main.py` или `docker compose up`
2. Откройте браузер: `http://localhost:8000/manage-prices`
3. Войдите под администратором (по умолчанию: `oilpress` / `MarshallJCM800`)
4. Редактируйте цены прямо в таблице или добавляйте новые товары

### Через Telegram-бота:

1. Запустите бота: `python bot.py` или `docker compose up bot`
2. Отправьте команду `/admin`
3. Нажмите `✏️ Управление ценами`
4. Выберите товар для редактирования или добавьте новый

---

## 💡 Советы по использованию

1. **Быстрое обновление цен:** Используйте веб-интерфейс для массового редактирования
2. **Оперативная коррекция:** Бот удобен для быстрого изменения цены "на ходу"
3. **Проверка перед добавлением:** Убедитесь, что название масла совпадает с номенклатурой
4. **Резервное копирование:** Перед массовыми изменениями экспортируйте базу данных

---

## 🔗 Интеграция с Avito

Ручное управление ценами **не конфликтует** с автоматическим обновлением из Avito:

- Кнопка `💰 Обновить цены из Avito` в боте работает как раньше
- При обновлении из Avito перезаписываются только найденные позиции
- Ручные изменения сохраняются до следующего автообновления

**Рекомендуемый рабочий процесс:**
1. Первоначальная настройка цен — вручную через веб/бота
2. Ежедневное обновление — автоматически из Avito
3. Экстренная коррекция — вручную через бота

## ❓ Вопросы и ответы

**Q: Можно ли восстановить удаленную позицию?**  
A: Нет, удаление необратимо. Добавьте товар заново через форму.

**Q: Что будет, если ввести некорректную цену?**  
A: Система покажет ошибку и запросит ввод повторно.

**Q: Синхронизируются ли цены между вебом и ботом?**  
A: Да, оба интерфейса работают с одной базой данных.

**Q: Нужно ли перезапускать приложение после изменения цен?**  
A: Нет, изменения применяются мгновенно.
