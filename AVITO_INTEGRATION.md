# Обновление цен из Avito

## Описание

Добавлен функционал автоматического получения актуальных цен с площадки Avito. 
Это основной канал продаж, поэтому цены должны быть всегда актуальными.

## Как это работает

1. **Парсинг Avito**: Модуль `AvitoParser.py` выполняет поиск товаров на Avito по названию масла и объему
2. **Извлечение цен**: Из найденных объявлений извлекаются цены
3. **Расчет средней цены**: Берется средняя цена из первых 3 подходящих результатов
4. **Обновление БД**: Цены в базе данных обновляются на найденные значения

## Использование

### Через Telegram-бота (рекомендуется)

1. Отправьте боту команду `/admin`
2. Нажмите кнопку **"💰 Обновить цены из Avito"**
3. Бот покажет статус обновления и количество обновленных записей

### Программно

```python
from AvitoParser import update_prices_from_avito

# Запуск обновления цен
updated_count = await update_prices_from_avito()
print(f"Обновлено записей: {updated_count}")
```

### Через командную строку

```bash
python AvitoParser.py
```

## Важные замечания

1. **Структура Avito**: Avito может менять верстку сайта, что потребует обновления парсера
2. **Лимиты запросов**: При большом количестве товаров могут быть ограничения со стороны Avito
3. **Точность поиска**: Не все товары могут быть найдены, если названия отличаются от указанных в каталоге
4. **Регион поиска**: По умолчанию поиск идет по всей России, можно добавить фильтрацию по региону

## Настройка

При необходимости можно настроить:
- Регион поиска (добавив параметр в URL)
- Количество обрабатываемых результатов (сейчас первые 10)
- Допустимую погрешность объема (сейчас ±500мл)
- Количество результатов для расчета средней цены (сейчас 3)
